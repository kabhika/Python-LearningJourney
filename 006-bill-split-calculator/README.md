**Date:** December 2024

- Formatting currency output with f-strings
- Creating user-friendly output displays

## Concepts Covered
- Arithmetic operations (multiplication, division)
- Percentage calculations
- Formatted output alignment
- Sequential program flow

## How It Works
1. User enters the total bill amount
2. User enters the number of people splitting the bill
3. User enters the desired tip percentage
4. Program calculates tip, grand total, and per-person amount

## How to Run
```bash
python bill_splitter.py
```

## Example Usage
```
=== Restaurant Bill Splitter ===

Enter the total bill amount: $85.50
Enter the number of people: 4
Enter the tip percentage (e.g., 15, 18, 20): 18

----------------------------------------
BILL SUMMARY
----------------------------------------
Subtotal:           $85.50
Tip (18%):          $15.39
Grand Total:        $100.89
----------------------------------------
Split 4 ways:       $25.22 per person
----------------------------------------
```
