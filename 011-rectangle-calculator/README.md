# Rectangle Calculator

**Date:** December 2024

## What I Learned
- Applying mathematical formulas in Python
- Order of operations with parentheses
- Type conversion from input strings to integers
- Clean output formatting

## Formulas Used
```
Area = length x width
Perimeter = 2 x (length + width)
```

## Concepts Covered
- Basic geometry calculations
- Arithmetic operators
- String concatenation for output
- Formatted display

## How to Run
```bash
python rectangle_calculator.py
```

## Example Usage
```
=== Rectangle Calculator ===

Enter the length of the rectangle: 10
Enter the width of the rectangle: 5

-----------------------------------
RECTANGLE PROPERTIES
-----------------------------------
Length:    10 units
Width:     5 units
-----------------------------------
Area:      50 square units
Perimeter: 30 units
-----------------------------------
```
