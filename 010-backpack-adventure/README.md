# Backpack Adventure - List Operations

**Date:** December 2024

## What I Learned
- Creating empty lists
- `append()` - add single item to end
- `extend()` - add multiple items from another list
- `remove()` - remove item by value
- `pop()` - remove item by index (returns the removed item)
- `insert()` - add item at specific position
- `index()` - find position of an item
- `len()` - count items in a list
- `set()` - convert to set for unique items
- List slicing with `[start:end]`

## List Methods Summary
| Method | What It Does |
|--------|--------------|
| `list.append(x)` | Add x to end |
| `list.extend([a,b])` | Add multiple items |
| `list.insert(i, x)` | Insert x at position i |
| `list.remove(x)` | Remove first occurrence of x |
| `list.pop(i)` | Remove and return item at position i |
| `list.index(x)` | Return position of x |
| `list[i]` | Access item at position i |
| `list[a:b]` | Slice from a to b |

## Concepts Covered
- List creation and manipulation
- All major list methods
