## BMI Formula
```
BMI = weight(kg) / height(m)^2
```

## BMI Categories (WHO Standard)
| BMI Range | Category |
|-----------|----------|
| < 18.5 | Underweight |
| 18.5 - 24.9 | Normal weight |
| 25 - 29.9 | Overweight |
| >= 30 | Obese |
