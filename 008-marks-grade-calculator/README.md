## Concepts Covered

## Grading Scale
| Percentage | Grade | Remark |
|------------|-------|--------|
| 90-100% | A | Excellent! |
| 80-89% | B | Very Good! |
| 70-79% | C | Good |
| 60-69% | D | Satisfactory |
| Below 60% | F | Needs Improvement |

## How to Run
```bash
python grade_calculator.py
```
